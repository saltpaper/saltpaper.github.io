什么是FPS

FPS: Frame Per Second (一秒钟渲染多少帧 )

CADisplayLink

YYKit中的FPSLabel 中用到了 CADisplayLink 来监视 CPU 的卡顿问题，看了代码后，自己想用swift 实现下。

首先查看CADisplaylink的官方文档

CADisplaylink: A timer object that allows your application to synchronize its drawing to the refresh rate of the display.

可以看出CADisplaylink 是一个计时器对象，允许应用程序使用这个对象将绘制与显示刷新频率同步。

3. preferredFramesPerSecond (优选帧速率)

这个属性也是iOS10之后才出现的，官方定义：

The frame rate that the display link uses to notifies the target.

说明帧率可以通过这个属性自己控制，但是实际的帧率和你设置的值可能有所不同，以下是对官方文档这部分的总结：

  1. 因为实际帧率受设备的最大帧速率控制的。如果设备最大帧速率为60，即使preferredFramesPerSecond = 70，结果fps也只是在60上下
  2. 优选的帧速率不是最大帧速率的除数，而是被舍入到最接近的因子，例如 preferredFramesPerSecond = 26 或 35，又如果设备最大帧速率为60，而实际帧率是30左右。
  3. 实际的帧率包括15，20，30 和 60（当设备最大帧速率为60时）。
